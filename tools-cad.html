Could my CAD become BIM?

We take your DWGs and do the following:

  • Create web friendly formats
    • DXF vector viewing in browser
    • PNGs of each paper layout in desired DPI
  • Extract info about all viewports
    • Scale
    • Transformation matrices
  • Formalise all relationships a knowledge graph

Knowledge graph subset

  • DWG -[alternative representation]-> DXF
  • DWG -[contains fragment]-> Paper layout
  • Paper layout -[contains fragment]-> Viewport
  • Viewport -[scale]-> 1:50
