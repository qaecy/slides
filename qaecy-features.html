CAD Document viewer

The CAD viewer shows DXF and DWG (through conversion). Layers can be toggled on/off and document fragments can be marked

The BIM viewer shows IFC models using That Open Company's IFC viewer

Our viewer includes features that utilize the underlying knowledge graph.

It will for example list all models available in the project.
